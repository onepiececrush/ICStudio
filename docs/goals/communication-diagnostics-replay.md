# 通信诊断与报文回放中心

## 目标

实现专业通信诊断中心，帮助用户定位现场通信问题。

## 页面 Tab

1. 实时报文
2. 通信统计
3. 异常诊断
4. 报文回放
5. 会话记录

## 实时报文字段

- 时间
- 方向：请求 / 响应
- 通道
- 协议：RTU / TCP
- 设备地址 / Unit ID
- 功能码
- 起始地址
- 数量
- 耗时
- 结果
- 异常码
- 原始报文
- 解析说明

## 通信统计

显示：

- 总请求数
- 成功数
- 失败数
- 超时数
- CRC 错误数
- 异常响应数
- 平均响应时间
- 最大响应时间
- 最慢设备
- 最容易失败地址段
- 最近 5 分钟成功率曲线

## 异常诊断

根据异常自动给出解释：

- 无响应：检查设备地址、串口线、IP、端口
- CRC 错误：检查波特率、校验位、线路干扰
- 异常码 01：非法功能码
- 异常码 02：非法地址
- 异常码 03：非法数据值
- 异常码 04：从站设备故障
- 响应长度异常：检查数据类型或寄存器数量

## 报文回放

支持：

- 选择历史报文
- 修改 Unit ID
- 修改地址
- 修改数据
- 重放到真实设备
- 重放到内置模拟器
- 连续回放
- 按原时间间隔回放
- 导出为 JSON/CSV

## 会话记录

一次连接过程保存为一个 session：

- 开始时间
- 结束时间
- 通信配置
- 报文数量
- 异常数量
- 关联工程
- 关联协议版本

## 验收标准

1. 所有主站请求和响应都能记录。
2. TCP 报文能解析 MBAP。
3. RTU 报文能校验 CRC。
4. 能按设备、功能码、地址范围筛选。
5. 能保存通信会话。
6. 能选择报文重新发送。
7. 能生成通信诊断摘要。
