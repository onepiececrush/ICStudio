# 组态页面生成器

## 目标

实现协议驱动的页面生成器，让用户可以从协议点位自动生成监控页面。

## 组件类型

支持：

- 数字卡片
- 状态灯
- 表格
- 仪表盘
- 趋势图
- 柱状图
- 告警列表
- 设备节点
- 能量流线
- 按钮
- 输入框

## 页面生成

从协议点位中根据 group/page/device_type 自动生成：

- PCS 实时数据页
- BMS 实时数据页
- 液冷页
- 动环页
- 电表页
- 箱变页
- 首页摘要页

## 拖拽设计器

左侧：组件库  
中间：画布  
右侧：属性面板  
底部：点位绑定和事件脚本

## 点位绑定

每个组件可以绑定：

- 设备实例
- 点位地址
- 显示格式
- 单位
- 枚举
- 状态颜色规则
- 告警规则

## 保存格式

保存为 JSON：

- page_id
- page_name
- layout
- widgets
- bindings
- actions
- styles

## 验收标准

1. 能从 PCS 协议生成实时数据页面。
2. 能拖拽组件。
3. 能绑定点位。
4. 能保存和重新打开页面。
5. 能和实时通信数据联动。
6. 能在自测模式下显示模拟数据。
