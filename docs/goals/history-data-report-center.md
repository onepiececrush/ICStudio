# 历史数据与报表中心

## 目标

实现本地历史数据记录和报表功能。

## 数据类型

需要记录：

- 实时点位数据
- 首页核心 KPI
- PCS 模块状态
- BMS 数据
- 液冷数据
- 动环数据
- 电表数据
- 箱变数据
- 告警事件
- 通信统计
- 操作日志
- 自动化测试结果

## 存储

第一阶段使用本地 SQLite。

表：

- point_history
- alarm_history
- communication_history
- operation_log
- test_report
- report_template

## 页面

Tab：

1. 趋势查询
2. 告警历史
3. 通信历史
4. 测试报告
5. 报表模板

## 趋势查询

筛选条件：

- 设备
- 点位
- 时间范围
- 采样周期
- 聚合方式：原始值 / 平均 / 最大 / 最小

图表：

- 趋势曲线
- 柱状图
- 表格

## 报表

支持：

- CSV 导出
- Excel 导出
- HTML 报告
- PDF 可后续实现

报告模板：

- 设备调试报告
- 自动化测试报告
- 告警统计报告
- 通信质量报告
- 首页自测报告

## 验收标准

1. 实时点位可以写入数据库。
2. 可以查询任意点位趋势。
3. 可以导出 CSV。
4. 告警历史可以查询。
5. 自动化测试结果可以生成报告。
6. 首页自测结果可以生成报告。
